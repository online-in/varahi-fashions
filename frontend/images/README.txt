Add your real product and banner photos here, for example:

  product-placeholder-1.jpg   (or rename products.json image paths to match)
  product-placeholder-2.jpg
  product-placeholder-3.jpg
  product-placeholder-4.jpg
  product-placeholder-5.jpg
  product-placeholder-6.jpg
  craft-loom.jpg              (used on the Home and About pages)

Until you add real images, the site automatically falls back to
placeholder graphics (via onerror="this.src='https://placehold.co/...'")
so nothing looks broken.

To update a product photo:
1. Drop the image file in this folder (e.g. saree-red-pattu.jpg)
2. In the Admin Dashboard, edit that product and set
   "Image path" to: images/saree-red-pattu.jpg
